# Pytobot

## Overview

Pytobot is a simple but powerful rpa automation tool designed to help you streamline your work processes and increase efficiency. Pytobot is easy to use, free, and can be used offline. It is also free for commercial use, making it great solution for businesses of all sizes. No coding experience is necessary and it works out-of-the-box.

## Installation

### Exe File

- Download the pytobot.exe file (don't need to download any other files)
- Double-click

### Clone Repo

- Clone the repo
- Install the libraries ("pip install -r requirements.txt")
- Run pytobot.py

## Usage

- Click on "Sample" to see a sample code
- Click on "Help" to see all available operators
- You can save the files you generate (.txt) and load them the next time you start the program
- Once you click on start - pytobot will start and not stop before the job is done. You can interrupt the process using the task manager.
